CheapStack

Moonshine.CheapStack — Type
struct CheapStack{T}

Simple stack container.

Functionalities

The following operations are supported:

  • isempty
  • empty!
  • first
  • length
  • pop!
  • push!

For convenience, CheapStack implements the iteration interface.
